    30. Conditional Statements


  30. Conditional statements are used to perform operations based on some conditions. 
    
                    If Statement

                    The block of code to be executed, when the condition specified is true.
                    if (condition) {
                    // block of code to be executed if the condition is true
                    }

                    If-else Statement

                    If the condition for the if block is false, then the else block will be executed.

                    if (condition) {
                    // block of code to be executed if the condition is true
                    }

                    else {
                    // block of code to be executed if the condition is false
                    }

                    Else-if Statement

                    A basic if-else ladder
                    if (condition1) {
                    // block of code to be executed if condition1 is true
                    }
                     else if (condition2) {
                    // block of code to be executed if the condition1 is false and condition2 is 
                    }
                     else {
                    // block of code to be executed if the condition1 is false and condition2 is 
                    }
                    
                    Switch Statement

                    Switch case statement in JavaScript

                    switch(expression) {
                    case x:
                    // code block
                    break;
                    case y:
                    // code block
                    default:
                    break;
                    // code block
                    }
